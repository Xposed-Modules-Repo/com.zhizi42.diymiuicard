# DIY MIUI NFC卡片卡面
自定义MIUI NFC卡片的卡面 可使用本地文件或http URL。支持官方MIUI未知-Hyper OS最新版（仅小米智能卡版本未知-24.07.23.1.f，更新版本不代表不支持但可能失效。其它小米系统使用相同的小米智能卡理论也支持）。
